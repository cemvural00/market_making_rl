## Overview

This document presents PnL (Profit and Loss) distribution plots for each agent in each environment. Each plot shows:

- **Histogram + KDE:** Distribution of PnL values across evaluation episodes
- **Mean (Red dashed line):** Average PnL across all episodes
- **Median (Green dashed line):** Median PnL value
- **IQR (Yellow shaded region):** Interquartile range (25th to 75th percentile)
- **Statistics box:** Summary statistics including mean, median, IQR, std dev, and sample size
